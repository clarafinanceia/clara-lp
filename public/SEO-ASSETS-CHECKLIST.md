# SEO Assets Checklist

Os seguintes arquivos precisam ser criados e adicionados à pasta `/public`:

## Imagens Open Graph (Obrigatório)
- [ ] `og-image.png` - 1200x630px - Imagem de compartilhamento social

## Favicons (Obrigatório)
- [ ] `favicon.ico` - 32x32px
- [ ] `favicon.svg` - SVG do logo
- [ ] `apple-touch-icon.png` - 180x180px

## PWA Icons (Recomendado)
- [ ] `icon-192.png` - 192x192px
- [ ] `icon-512.png` - 512x512px

## Dicas para og-image.png
- Dimensões: 1200x630px
- Inclua o logo Clara
- Use as cores da marca (#1b329a, #728fbf)
- Texto legível: "Clara - Sua IA de Finanças Pessoais"
- Pode usar uma captura estilizada da landing page

## Ferramentas úteis
- Favicon Generator: https://realfavicongenerator.net/
- OG Image Preview: https://metatags.io/
- PWA Icon Generator: https://www.pwabuilder.com/imageGenerator

## Após criar os assets
1. Adicione os arquivos na pasta `/public`
2. Delete este arquivo (SEO-ASSETS-CHECKLIST.md)
3. Teste com: https://metatags.io/
